The PS2 in action

Modbo 5.0D, Pic Fix, and IDE ressurector

The Modbo allows for imported games to be played as well as add a custom bios.
This bios alllows for unsigned code to run as well as boot into custom options.
The Pic Fix will turn off the console if it dectects the laser will crash a specfic
chip on the board that has the potential to burn it out. The IDE ressurector
Rebuilds the IDE hard drive circuit that was left ofver during the first PS2 Slim
revision. This allows you to get an IDE to SD adapter and wire up a micro usb.
From there, you can put games on the SD card so in the event the drive fails, you can
still play games as long as you have a backup of them.

NeoGeo CD

NeoGeo CD SD Loader

This allows the NeoGeo CD to run games off a micro usb.
this is useful as it loads 4x faster than the regular CD drive.
also has the added benefit of changing regions.
